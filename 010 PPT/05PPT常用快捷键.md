# PPT常用快捷键

### 常规处理

| 快捷键           | 功能             |
| ---------------- | ---------------- |
| ctrl + n         | 新建一个PPT      |
| ctrl + o         | 打开一个PPT      |
| ctrl + s         | 保存             |
| ctrl + shift + s | 另存为           |
| ctrl + c         | 复制             |
| ctrl + v         | 粘贴             |
| ctrl + z         | 撤回             |
| ctrl + y         | 恢复撤回前的操作 |
