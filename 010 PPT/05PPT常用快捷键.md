# PPT常用快捷键

### 常规处理

| 快捷键           | 功能             |
| ---------------- | ---------------- |
| ctrl + n         | 新建一个PPT      |
| ctrl + o         | 打开一个PPT      |
| ctrl + s         | 保存             |
| ctrl + shift + s | 另存为           |
| ctrl + c         | 复制             |
| ctrl + v         | 粘贴             |
| ctrl + z         | 撤回             |
| ctrl + y         | 恢复撤回前的操作 |

### 字体处理

| 快捷键                     | 功能               |
| -------------------------- | ------------------ |
| ctrl + [(ctrl + shift + ,) | 减小字号           |
| ctrl + ](ctrl + shift + .) | 增大字号           |
| ctrl + b                   | 加粗               |
| ctrl + i                   | 倾斜               |
| ctrl + u                   | 下划线             |
| ctrl + l/e/r               | 文本对齐(左/中/右) |

### 通用处理

| 快捷键                  | 功能                     |
| ----------------------- | ------------------------ |
| ctrl + a                | 全选                     |
| F4                      | 重复上一步操作           |
| ctrl + d                | 复制并粘贴               |
| shift + 拖动对象        | 沿水平(或垂直)线拖动对象 |
| ctrl + shift + 拉伸元素 | 按中心缩放               |
| shift + 拉伸元素        | 等比例缩放               |
| ctrl + shift + c        | 复制格式                 |
| ctrl + shift + v        | 粘贴格式                 |
| ctrl + g                | 组合                     |
| ctrl + shift + g        | 取消组合                 |

### 画布处理

| 快捷键          | 功能            |
| --------------- | --------------- |
| F5              | 从头放映PPT     |
| shift + F5      | 从当前面放映    |
| alt + F9        | 显示/隐藏参考线 |
| shift + F9      | 显示/隐藏网格线 |
| ctrl + 鼠标滚轮 | 快速缩放画布    |
