## 运算符 - 优先级可以通过()调整

| ------ | -------------------------------------------- | ------------------------------ |
| 13     | `-、+`                                       | 单目负、单目正                 |
| 12     | `!、~`                                       | 逻辑非、按位取反或补码         |
| 11     | `*、/、%`                                    | 乘、除、取模                   |
| 10     | `+、-`                                       | 加、减                         |
| 9      | `<<、>>`                                     | 按位左移、按位右移             |
| 8      | `<=、>=、<、>`                               | 小于等于、大于等于、小于、大于 |
| 7      | `==、!=`                                     | 等于、不等于                   |
| 6      | `&`                                          | 按位于                         |
| 5      | `^`                                          | 按位异或                       |
| 4      | &#124;                                       | 按位或                         |
| 3      | `&&`                                         | 逻辑与                         |
| 2      | &#124;&#124;                                 | 逻辑或                         |
| 1      | =、+=、-=、*=、/=、&=、^=、&#124;=、<<=、>>= | 赋值、运算且赋值               |
