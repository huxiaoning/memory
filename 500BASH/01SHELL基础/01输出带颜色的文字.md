# 输出带颜色的文字

```bash
# 输出红色的Hello World
$ echo -e "\e[1;31mHello World\e[0m"
Hello World
```





| 符号  | 颜色 |
| :---: | :--: |
| `30m` | 黑色 |
| `31m` | 红色 |
| `32m` | 绿色 |
| `33m` | 黄色 |
| `34m` | 蓝色 |
| `35m` | 洋红 |
| `36m` | 青色 |
| `37m` | 白色 |

## `echo`命令`-e`参数识别的控制字符

| 控制字符 | 作用                                                         |
| :------: | ------------------------------------------------------------ |
|   `\a`   | 输出警告音                                                   |
|   `\b`   | 退格键，也就是向左删除键                                     |
|   `\n`   | 换行符                                                       |
|   `\r`   | 回车键                                                       |
|   `\t`   | 制表符，也就是`Tab`键                                        |
|   `\v`   | 垂直制表符                                                   |
| `\0nnn`  | 按照八进制`ASCII`码表输出字符。其中0为数字零，`nnn`是三位八进制数 |
|  `\xhh`  | 按照十六进制`ASCII`码表输出字符。其中`hh`是两位十六进制数    |
