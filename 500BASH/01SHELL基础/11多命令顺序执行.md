# 多命令执行顺序



| 多命令执行符 | 格式             | 作用                                       |
| ------------ | ---------------- | ------------------------------------------ |
| `;`          | 命令1;命令2      | 多个命令顺序执行，命令之间没有任何逻辑关系 |
| `&&`         | 命令1 && 命令2   | 逻辑与，当命令1正确执行后，命令2才会执行   |
| `||`         | 命令1 \|\| 命令2 | 逻辑或，当命令1执行不正确，命令2才会执行   |
