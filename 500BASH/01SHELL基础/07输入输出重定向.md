# 输入输出重定向



## 输出重定向

| 类型               | 符号              | 作用                     |
| ------------------ | ----------------- | ------------------------ |
| 标准输出重定向     | 命令 > 文件       | 正确输出，覆盖指定文件   |
| 标准输出重定向     | 命令 >> 文件      | 正确输出，追加到指定文件 |
| 标准错误输出重定向 | 错误命令 2> 文件  | 错误输出，覆盖指定文件   |
| 标准错误输出重定向 | 错误命令 2>> 文件 | 错误输出，追加到指定文件 |



## 输入重定向

```bash
# 实现yum自动回答yes，相当于 yum install -y ....
# 原理是把原本需要从终端手动输入y,重定向到从文件输入y
$ vim yes.txt
y

$ yum install python < yes.txt
```
